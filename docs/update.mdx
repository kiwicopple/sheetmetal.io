Send `PATCH` or `PUT` request to update values of a row.

- `PUT` request updates whole row. If you do not provide all fields in the JSON object, some fields might get emptied. PUT returns whole updated rows.
- `PATCH` request updates only values passed in JSON. Returns only updated fields.

To identify the row(s) you want to update you need to add `/{column_name}/{value}` to the API URL. Then you need to pass JSON object with new values. Updates only rows where `{column_name}` match `{value}`.

### Update the value in one column

The `PATCH` method returns only updated values.

**Example**

Update all rows from sheet "Sheet1" where value of column name is Peter

```sh
curl "https://sheetsu.com/apis/v1.0db/020b2c0f/sheets/Sheet1/name/Peter" \
-X PATCH \
-H "Content-Type: application/json" \
-d '{ "score": "1337" }'

```

**Returns**

An array of updated objects.

### Update whole row

The `PUT` method returns whole updated rows.

**Example**

Update all rows from sheet "Sheet1" where value of column name is Lois

```sh
curl "https://sheetsu.com/apis/v1.0db/020b2c0f/sheets/Sheet1/name/Lois" \
-X PUT \
-H "Content-Type: application/json" \
-d '{ "id": "2", "name": "Loo1z", "score": "99999" }'

```

**Returns**

An array of updated objects.
