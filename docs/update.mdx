Before updating data from your Google Sheet you need login and connect SheetMetal to your Sheet. Each time you connect a Sheet we generate a private key.

Send `PATCH` or `PUT` request to update values of a row.

- `PUT` request updates whole row. If you do not provide all fields in the JSON object, some fields might get emptied. PUT returns whole updated rows.
- `PATCH` request updates only values passed in JSON. Returns only updated fields.

To identify the row(s) you want to update you need to add `/{column_name}/{value}` to the API URL. Then you need to pass JSON object with new values. Updates only rows where `{column_name}` match `{value}`.

### Returns

An array of updated objects. For example:

<pre>{EXAMPLE_RESPONSE}</pre>

### Examples

Update all rows from sheet "Employees" where value of "gender" is male":

<pre>{UPDATE_VALUE}</pre>

Update the entire row from sheet "Employees" where value of "id" is "1":

<pre>{UPDATE_ROWS}</pre>
