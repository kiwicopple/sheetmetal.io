Before getting data from your Google Sheet you need login and connect SheetMetal to your Sheet. Each time you connect a Sheet we generate a private key.

Use the generated key to send a `GET` request. For example, if you had a tab in your Google Sheet called "Employees":

<code>{GET_ALL}</code>

### Request Parameters

You can optionally limit results, or start with offset.

| Parameter | Description                                                |
| --------- | :--------------------------------------------------------- |
| limit     | Number of how many rows should be returned                 |
| offset    | Number from which row response should start (default is 0) |

### Returns

An array of objects. Each object is a row from the Google Spreadsheet. For example:

<pre>{EXAMPLE_RESPONSE}</pre>

### Examples

Get all rows from "Employees":

<pre>{GET_ALL}</pre>

Read first two rows from sheet "Employees":

<pre>{GET_SOME}</pre>

Read the third row from sheet "Employees":

<pre>{GET_OFFSET}</pre>
