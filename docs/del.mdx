export const metadata = {
  title: 'Delete',
}

export const BASE_URL = process.env.BASE_URL || 'https://sheetmetal.io'
export const API_URL = `${BASE_URL}/api/v1`
export const METAL_KEY = 'YOUR_METAL_KEY'
export const TAB_NAME = 'Employees'
export const CLEAR = `curl "${API_URL}/${METAL_KEY}/Employees/id/1" \\
-X DELETE
`
export const DESTROY = `curl "${API_URL}/${METAL_KEY}/Employees" \\
-X DELETE \\
-H "Content-Type: application/json" \\
-d '{ name": "Erlich Bachman", "status": "terminated" }'
`

Before deleting data from your Google Sheet you need login and connect SheetMetal to your Sheet. Each time you connect a Sheet we generate a private key.

- To _CLEAR_ a row, send a `DELETE` request _with_ a `/{column_name}/{value}` path
- To _DESTROY_ a row, send a `DELETE` request _without_ a `/{column_name}/{value}` path, but with a payload specifying the criteria

### Returns

On success, returns `204 No Content` HTTP status code, without a body.

### Examples

Clear all rows from sheet "Employees" where "id" is "1":

<pre>{CLEAR}</pre>

Destroy all rows from sheet "Employees" where column "name" is "Erlich Bachman" and column "status" is "terminated":

<pre>{DESTROY}</pre>
